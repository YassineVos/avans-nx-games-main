Gamesplatform Dashboard

Over dit Project

Dit platform is ontwikkeld voor het vak NoSQL binnen de opleiding Informatica aan Avans Hogeschool. De applicatie stelt gebruikers in staat om digitale games te bekijken, beoordelen en als favoriet op te slaan. De applicatie is gebouwd met een moderne webstack, bestaande uit:

  • NestJS backend met MongoDB en Neo4j
  • Angular frontend in een Nx-monorepo
  • JWT-gebaseerde authenticatie
  • Ondersteuning voor gebruikersfavorieten, reviews en gebruikersprofielen

De frontend en backend zijn gescheiden en communiceren via REST API's. De gegevens worden opgeslagen in MongoDB, en Neo4j wordt gebruikt voor sociale relaties (zoals vriendschappen en gedeelde favorieten).

Belangrijkste Functionaliteiten

  • Games toevoegen, bekijken, bewerken en verwijderen
  • Games beoordelen via reviews
  • Favoriete games opslaan en beheren
  • Overzicht van games per gebruiker
  • JWT-authenticatie en beveiligde endpoints

User Stories en Acceptatiecriteria

User Story 1 – Games bekijken

Als gebruiker wil ik een lijst van games kunnen zien zodat ik kan ontdekken welke games beschikbaar zijn.

  • De gebruiker ziet een overzicht van alle games
  • Elke game toont een titel, beschrijving en release-informatie

User Story 2 – Game toevoegen

Als gebruiker wil ik een nieuwe game kunnen toevoegen zodat ik mijn favoriete games kan delen met anderen.

  • Alleen ingelogde gebruikers kunnen een game toevoegen
  • Een game bevat minimaal een titel, beschrijving, genre, en spelersinformatie

User Story 3 – Game beoordelen

Als gebruiker wil ik een review kunnen achterlaten bij een game zodat ik mijn mening kan delen.

  • De gebruiker kan een review met een score (1–5) en optioneel een opmerking plaatsen
  • Alle reviews zijn zichtbaar per game

User Story 4 – Games als favoriet opslaan

Als gebruiker wil ik games als favoriet kunnen markeren zodat ik ze later makkelijk terug kan vinden.

  • Alleen ingelogde gebruikers kunnen games als favoriet markeren
  • Favoriete games zijn zichtbaar op een aparte pagina
